SSH公開鍵を追加

この鍵を識別するための名前
公開鍵の内容を貼り付けてください（ssh-rsa, ssh-ed25519, ecdsa形式に対応）
Windowsの場合: type %USERPROFILE%\.ssh\id_ed25519.pub
Mac/Linuxの場合: cat ~/.ssh/id_ed25519.pub
公開鍵を登録すると、パスワードなしでSSH接続できます。
鍵がない場合は ssh-keygen -t ed25519 で生成してください。
