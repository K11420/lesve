LenovoServerの使い方ガイド

概要

LenovoServerは、Ubuntuベースの仮想サーバー環境を提供するサービスです。 Discord Bot、Webアプリ、APIサーバーなど、様々なサービスを簡単にホスティングできます。

無料で利用可能
0.5 vCPU / 512MB メモリ / 1GB ストレージのサーバーを無料で利用できます。

使い始める

ステップ1: ログイン

Discordアカウントでログインします。

  1. ログインページにアクセス
  2. 「Discordでログイン」ボタンをクリック
  3. Discordで認証を許可

ステップ2: サーバー作成

  1. ダッシュボードで「新規作成」ボタンをクリック
  2. サーバー名を入力（英数字、ハイフン、アンダースコアのみ）
  3. サーバータイプを選択
  4. 「作成」をクリック

ステップ3: SSH接続

作成したサーバーにSSHで接続して、自由にカスタマイズできます。

ssh srv-XXXXXXXX@106.73.68.66 -p 4104

SSH接続

接続情報

ホスト 106.73.68.66
ポート 4104
ユーザー名 ダッシュボードで確認（srv-XXXXXXXX形式）
パスワード ダッシュボードで確認（クリックでコピー）

パスワード認証

ダッシュボードに表示されるパスワードを使用して接続します。

  1. ダッシュボードでサーバーを選択
  2. SSH接続情報を確認（パスワードはクリックでコピー可能）
  3. ターミナルで接続コマンドを実行
  4. パスワードを入力
# 接続コマンド
ssh srv-XXXXXXXX@106.73.68.66 -p 4104

# パスワードを聞かれるので入力
srv-XXXXXXXX@106.73.68.66's password: ********

SSH公開鍵認証（推奨）

パスワードなしで安全に接続できます。一度設定すれば毎回パスワードを入力する必要がありません。

1. 鍵の生成（持っていない場合）

# Windows (PowerShell) / Mac / Linux 共通
ssh-keygen -t ed25519

# 質問にはすべてEnterでOK

2. 公開鍵の確認

# Windows (PowerShell)
type $env:USERPROFILE\.ssh\id_ed25519.pub

# Mac / Linux
cat ~/.ssh/id_ed25519.pub

表示される ssh-ed25519 AAAA... で始まる1行をコピーします。

3. ダッシュボードで登録

  1. ダッシュボードの「設定」ページを開く
  2. 「SSH公開鍵」セクションで「鍵を追加」をクリック
  3. 名前を入力（例: My PC）
  4. 公開鍵の内容を貼り付け
  5. 「追加」をクリック

4. 接続テスト

ssh srv-XXXXXXXX@106.73.68.66 -p 4104

# パスワードを聞かれずに接続できれば成功！
ポイント
登録した公開鍵は、あなたの全サーバーで使用できます。 PCを複数お持ちの場合は、それぞれの公開鍵を登録してください。

設定

起動コマンド

サーバー起動時に自動実行されるコマンドを設定できます。

# 例: Node.jsアプリを起動
node app.js

# 例: Pythonスクリプトを起動
python3 bot.py

環境変数

Discord BotのトークンやAPIキーなど、秘密情報を環境変数として設定できます。

DISCORD_TOKEN=your_token_here
API_KEY=your_api_key

プログラムから process.env.DISCORD_TOKEN（Node.js）や os.environ['DISCORD_TOKEN']（Python）で取得できます。

自動再起動

有効にすると、プログラムがクラッシュした場合に自動的に再起動します。

よくある質問

無料で使えますか？
はい、無料でご利用いただけます。0.5 vCPU / 512MB メモリ / 1GB ストレージのサーバーを1台作成できます。
どんな用途に使えますか？
Discord Bot、Webアプリ、APIサーバー、Minecraftサーバー（軽量版）など、様々な用途にお使いいただけます。root権限があるので自由にカスタマイズ可能です。
24時間稼働できますか？
はい、サーバーは24時間365日稼働します。自動再起動機能を有効にすれば、クラッシュ時も自動的に復旧します。
SSH接続のパスワードを忘れました
ダッシュボードでサーバーを選択すると、SSH接続情報（ユーザー名・パスワード）を確認できます。パスワードはクリックでコピーできます。
毎回パスワードを入力するのが面倒です
SSH公開鍵認証を設定すると、パスワードなしで接続できます。設定ページで公開鍵を登録してください。詳しくは公開鍵認証をご覧ください。
ファイルをアップロードするには？
以下の方法があります：
  • ダッシュボードのファイルマネージャーを使用
  • Git経由でクローン（git clone）
  • SCP/SFTPでアップロード
Discordアカウントを持っていません
申し訳ありませんが、現在はDiscordアカウントでのログインのみ対応しています。こちらから無料でDiscordアカウントを作成できます。
サーバーを削除したらデータは消えますか？
はい、サーバーを削除するとすべてのデータが完全に削除され、復元できません。削除前に必要なデータはバックアップしてください。
ポート開放はできますか？
現在、外部からのポート開放には対応していません。Discord Botなど、アウトバウンド通信のみで動作するサービスに最適です。
どのプログラミング言語が使えますか？
Ubuntu環境なので、ほぼすべての言語が使えます。サーバータイプで Node.js、Python、Java、Go がプリインストールされたものを選べます。その他の言語も apt-get でインストール可能です。

トラブルシューティング

SSH接続でエラーが出る

「WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!」

以下のコマンドで古いホストキーを削除してください：

ssh-keygen -R "[106.73.68.66]:4104"

「Connection refused」

サーバーが停止している可能性があります。ダッシュボードでサーバーを起動してください。

SSH鍵認証ができない

確認ポイント
  • 接続元PC の公開鍵を登録していますか？（サーバー内で生成した鍵ではなく）
  • 登録後30秒以上待ちましたか？
  • サーバーは起動していますか？

コマンドが見つからない

必要なパッケージがインストールされていない場合は、以下でインストールできます：

apt-get update && apt-get install -y パッケージ名

プログラムが動かない

  • エラーメッセージを確認してください
  • 必要な依存関係がインストールされているか確認
  • 環境変数が正しく設定されているか確認
  • メモリ不足の場合は、軽量なプログラムを使用してください
